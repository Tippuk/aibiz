# 🎬 Zero-Cost Конвейер: TikTok Shorts (Темная Психология)

Этот документ содержит пошаговый план стабильной сборки автоматизированного генератора видео. Скрипт переписан на использование надежной библиотеке Gemini API (вместо g4f) и использует оптимизированную архитектуру рендера кадров для быстрого монтажа. В проект также добавлен графический интерфейс (GUI).

## 🛠 1. Подготовка окружения и зависимости

Для работы конвейера потребуется Python 3.10+ и библиотеки.

**Установка Python-зависимостей:**
```bash
pip install customtkinter google-generativeai edge-tts requests faster-whisper moviepy==1.0.3 numpy
```

**Дополнительное системное ПО (Обязательно для Windows):**
1. Скачайте и установите **ImageMagick** (https://imagemagick.org/script/download.php#windows).
2. При установке **обязательно** поставьте галочку `Install legacy utilities (e.g. convert)`.
3. Запомните путь установки (обычно `C:\Program Files\ImageMagick-7.X.X-Q16-HDRI\magick.exe`).
4. Отредактируйте файл `config_defaults.py` в папке с установленной библиотекой `moviepy` (находится в `Lib\site-packages\moviepy\`), изменив параметр `IMAGEMAGICK_BINARY` на путь к `magick.exe`.

---

## 🏗 2. Архитектура проекта

Комплекс состоит из двух основных файлов:
1. `app.py` — Графический интерфейс для ввода ключей и тем, запускает фонерную генерацию.
2. `auto_tiktok_pipeline.py` — Ядро конвейера. Генерирует текст (Gemini), озвучку (edge-tts), скачивает фон (Pexels API), сводит музыку (из папки `dark_music`) и монтирует видео (MoviePy).

---

## 💻 3. Запуск конвейера (Только консоль)

Если вы хотите запустить ядро напрямую без интерфейса, отредактируйте блок `if __name__ == "__main__":` в самом низу файла `auto_tiktok_pipeline.py`, вписав туда свои API ключи, после чего запустите скрипт:
```bash
python auto_tiktok_pipeline.py
```

---

## 🖥 4. Запуск с Графическим Интерфейсом (GUI)

Для удобной работы создан десктопный интерфейс на базе **CustomTkinter** с темной темой.

**Преимущества GUI:**
* Удобный интерфейс для ввода API ключей Pexels и Gemini.
* Ключи автоматически сохраняются в локальный файл `config.json` и восстанавливаются при сделующем запуске.
* Большое текстовое поле для массового ввода тем (каждая тема с новой строки).
* Live-консоль (только для чтения), в которую выводятся все логи и статус сборки видео.
* Безопасная многопоточность: генерация роликов идет в фоне, интерфейс программы не зависает (`Not Responding`) во время рендера видео.

**Инструкция по запуску:**
1. Поместите пару мрачных треков в папку `dark_music` в корне проекта (иначе видео соберется без музыки).
2. Запустите консоль в папке проекта и выполните команду:
```bash
python app.py
```
3. Откроется стильное черное окно.
4. Введите ваши API-ключи слева и нажмите **Save Keys**.
5. Заполните список тем справа вверху.
6. Нажмите кнопку **🚀 Запуск конвейера**.
7. Наблюдайте за процессом в окне "Console Output". Готовые видео появятся в папке `output_videos`.
