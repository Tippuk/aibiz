# TikTok Automation Pro 🚀

AI-система для полной автоматизации создания и публикации коротких видео (TikTok/Shorts) для четырех ниш: Психология, Финансы, Истории и Документалистика.

## 🌟 Основные возможности
- **AI-Генерация**: Автоматическое создание сценариев через Groq (Llama-3).
- **Многонишевость**: Поддержка 4 разных каналов с уникальными промптами и голосами.
- **Интеллектуальный монтаж**: Автоматический подбор видео (Pexels), озвучка (Edge-TTS), наложение музыки и субтитров.
- **Shadow DOM Uploader**: Умный загрузчик для TikTok Studio, пробивающий любую защиту.
- **Reddit Parser**: Автоматический парсинг интригующих историй с топовых сабреддитов.

## 🛠 Предварительные требования
Перед установкой убедитесь, что у вас установлены:
1. **Python 3.10+**
2. **FFmpeg**: (Добавьте в PATH)
3. **ImageMagick**: Необходим для работы субтитров.
   - [Скачать ImageMagick](https://imagemagick.org/script/download.php) (рекомендуется версия Q8 или Q16).
   - При установке **обязательно** отметьте галочку `Install legacy utilities (convert, etc.)`.

## 📦 Установка

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/твой-аккаунт/aibiz.git
   cd aibiz
   ```

2. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Инициализируйте Playwright**:
   ```bash
   playwright install chromium
   ```

## ⚙️ Настройка

### 🔑 Получение ключей
1. **Groq API Key**: Зарегистрируйтесь на [Groq Cloud](https://console.groq.com/) и получите бесплатный ключ. Он используется для написания сценариев.
2. **Pexels API Key**: Получите ключ на [Pexels API](https://www.pexels.com/api/). Он нужен для скачивания фоновых видео.

### 📝 Конфигурация
Запустите приложение `app.py`, перейдите в раздел **Конфигурация API** и вставьте ваши ключи. Настройки сохранятся в файл `config.json`.

**Важно (ImageMagick)**:
Откройте `auto_tiktok_pipeline.py` и проверьте путь на 3-й строке:
```python
os.environ["IMAGEMAGICK_BINARY"] = r"C:\Program Files\ImageMagick-7.1.2-Q8\magick.exe"
```
Убедитесь, что путь соответствует вашей папке установки ImageMagick.

## 🎬 Запуск

### Запуск через GUI (Рекомендуется)
```bash
python app.py
```
В интерфейсе вы сможете:
1. Выбрать нишу.
2. Сгенерировать темы или спарсить их с Reddit.
3. Запустить конвейер создания и загрузки видео.
4. Отслеживать прогресс в умной консоли.

### Настройка TikTok профилей
Перед первым запуском конвейера необходимо сохранить сессии входа для каждой ниши:
1. В GUI нажмите на статус аккаунта в сайдбаре или запустите скрипт:
   ```bash
   python tiktok_uploader.py login psychology
   ```
2. Откроется окно браузера — залогиньтесь в ваш аккаунт TikTok.
3. Дождитесь закрытия окна. Сессия сохранится в файл `state_psychology.json`. Повторите для всех ниш.
